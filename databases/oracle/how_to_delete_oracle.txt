
 Delete Oracle* registry keys
Press Windows key, type regedit, delete the following Windows registry keys if it exists. :

2.1 HKEY_LOCAL_MACHINE/SOFTWARE/Oracle*
HKEY_LOCAL_MACHINE/SOFTWARE/Oracle*
HKEY_LOCAL_MACHINE/SOFTWARE/Wow6432Node/Oracle* (64-bits Windows)
HKEY_LOCAL_MACHINE/SYSTEM/CurrentControlSet/Services/Oracle*
